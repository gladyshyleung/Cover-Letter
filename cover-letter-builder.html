✕ iCloud Drive setup (do once)
  1. On your Mac, move this HTML file into iCloud Drive — e.g. a folder called Cover Letter Builder.
  2. On iPhone, open the Files app → iCloud Drive → find the file → tap to open in Safari.
  3. In Safari, tap Share → Add to Home Screen for quick access.
  4. Each time you make changes, use Save to iCloud below to keep both devices in sync.
